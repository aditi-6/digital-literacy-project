## 📌 Project Overview
This repository contains my Digital Literacy Portfolio created as part of the Digital Literacy course. The objective of this project is to develop awareness and practical understanding of essential digital skills, including the use of online tools, professional communication, collaboration platforms, and cyber safety.

As a Student Digital Ambassador, this project reflects my efforts to guide peers in navigating the digital world responsibly and effectively.
